Como correr a aplicação web:
1- Instalar o node
	- Aceder ao site e fazer download da aplicação: https://nodejs.org/en/
	
2 - Instalar o react:
	- Na consola do windows escrever: npm init
									  npm install --save react react-dom

4 - Instalar o git:
	- Aceda ao site do git e faça download da aplicação: https://git-scm.com/downloads
									  
5 - Fazer download do repositorio git:
	- Escolha uma pasta onde quer guardar o projeto.
	- Na consola aceda a essa pasta e de seguida escreva: git clone https://github.com/relense/landingPage.git
	
6 - Modulos de react:
	- Agora que tem o projeto instalado aceda à sua pasta na consola e escreva também nesta: npm install --save
	
7 - Neste momento o site está pronto a correr localmente:
	- Na consola dentro da pasta do projeto escreva: npm start
	
8 - Uma página irá abrir no browser e nesta poderá vera aplicação.
9 - Caso não seja aberta a página no browser pode aceder através de: localhost:3000
